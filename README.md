# 📸 eXposure – Photography Portfolio Website

A modern, responsive photography portfolio website built with React and Tailwind CSS.  
Designed for impactful visual storytelling, smooth animations, and clean user interaction.

## 📂 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Mantochi/photograph.git
cd photograph
npm install
npm start
```

Build for production:

```bash
npm run build
```

Deployment:

Deploy using GitHub Pages

```bash
npm run deploy
```
